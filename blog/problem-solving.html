Problem Solving
  1. When I was blocked on a simple problem.

    I was blocked when trying to figure out the WhereAreYou builtin method. My issue wasn't clear to me, I assumed that I wasn't referring to the array correctly so I looked up documentation around that. Tried a few alternate methods that all failed. Carefully looked through my code with console logs to see where I was going wrong. Took a break to try with a fresh head. Frustrated I gave up and moved on. It wasn't until I was sent my code to a teacher and he said it should work fine did I try run it again. Perhaps I just needed to reset VSCode or my computer, or just run it a second time to confirm.

  2. Elegantly solved a problem

    While doing a Kata on Code Wars I was extremely satisfied to provide a 2 sentence answer to a problem. It was take an array of numbers, and sort them into ascending order. I had to research the documentation a fair bit, take a few breaks and refactor a couple of parts. But I eventually arrived at
    let array = n.toString().split('').sort().reverse().join('') return parseInt(array)
    Which looks and felt great to get to. There is definitely a way to have a single return line but this is close enough for me for now.
